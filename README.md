# HelloReady (Working Title)

A mobile-first phrase card app that helps people communicate in unfamiliar situations.

The first phrase pack focuses on weddings, helping non-native speakers quickly listen to, practice, and use common phrases during social interactions.

## Features

- Tap-to-hear pronunciation
- Traditional Chinese + English phrase cards
- Phonetic pronunciation support
- Edit and customize phrases
- Mobile-first design
- Browser-based storage
- No account required

## Current Theme

- Wedding

## Future Themes

- Restaurant
- Shopping
- Travel

## Technology

- HTML
- CSS
- Vanilla JavaScript
- Web Speech API
